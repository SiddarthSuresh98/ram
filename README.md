# Risc V[ECTOR]

## dependencies
- cmake
- g++ (GCC) 11.4.0
- python 3.10 or newer
- catch2 version 3.5.3
- Qt version 6.8.2

## to compile
Generate the build directory with

`cmake -S . -B build`

then compile both the simulator and tests with

`cmake --build build`
